## Fallodの使い方
completed formフォルダにあるLteHttpSecureClient2.inoをMainCore　

Sub1フォルダにあるSub1.inoをSubcore1で動作させてください。

最後にスライドスイッチをONにすれば起動します。

GPS機能が動くまでに多少時間がかかるのでスイッチ起動後、30秒程度停止して下さい。
